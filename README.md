# Sales-Dashboard-PowerBI
# 📊 Sales Dashboard using Power BI

## Overview
This project is a Sales Dashboard developed using Microsoft Power BI to analyze sales performance and provide business insights through interactive visualizations.

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Sales Trend Analysis
- Category-wise Sales
- Region-wise Sales
- Interactive Filters (Slicers)
- Dynamic Charts and Graphs

## Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
